1. buat database terlebih dahulu. 
   tabel yang dibuat dapat mengikuti petunjuk yang ada 
   di folder database.
2. lengkapi method yang ada di setiap controller (auth dan dashboard)
3. sesuaikan konfigurasi database di model
4. lengkapi method yang ada di setiap model (user dan student)
5. index adalah file utama yang dijalankan saat mengakses web
6. folder exceptions digunakan untuk exception handling secara khusus
7. folder controllers digunakan untuk controller
8. folder models digunakan untuk model
9. folder views digunakan untuk view
   a. auth untuk tampilan login dan register
   b. dashboard untuk tampilan dashbord (index), profile, dan data siswa (students)
      (i) index untuk tampilan seluruh data siswa
      (ii) create untuk tampilan tambah data siswa
      (iii) edit untuk tampilan ubah data siswa
   c. errors untuk tampilan error
   d. layouts untuk template tampilan lainnya
      (i) header untuk template element header (HTML5)
      (ii) footer untuk template element footer (HTML5)
      (iii) auth untuk template login dan register disertai element footer
      (iv) main untuk template halaman setelah login 
           disertai element header dan footer
      ** template digunakan agar header dan footer tidak redundan
10. unggah hasil kode program dan database di hosting dan domain gratis
    (misal: https://www.infinityfree.com/)
11. hati-hati dengan huruf case-sensitive
